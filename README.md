# Dojo - Proyecto Caja Login

## Sprint 1

## Descripción
Este proyecto es una aplicación web simple que implementa una página de login básica. Estamos en el primer sprint del desarrollo, enfocándonos en la creación de la interfaz de usuario y la funcionalidad básica de autenticación.

## Sprint 1: Objetivos
- Crear la estructura HTML de la página de login.
- Preparar la base para futuras funcionalidades de autenticación.

## Características
- Página de login con campos para usuario y contraseña.
- Interfaz de usuario intuitiva.

## Instalación
1. Clona este repositorio.
2. Abre el archivo `login.html` en tu navegador web.

## Tecnologías Utilizadas
- HTML

## Sprint 2

## Descripción
Este proyecto es una aplicación web que implementa una página de login con diseño moderno, responsivo y adaptado a dispositivos móviles y de escritorio. Incluye un avatar personalizado, estilos en tonos de azul petróleo, colores invertidos para un look único, y una interfaz centrada y profesional. Estamos en el segundo sprint del desarrollo, avanzando en la mejora de la interfaz de usuario y la preparación para funcionalidades avanzadas de autenticación.

## Sprint 2: Objetivos
- Implementar diseño responsivo y centrado para móviles y desktop.
- Agregar estilos CSS externos en tonos de azul petróleo.
- Incluir un avatar SVG sobre la caja de login.
- Optimizar la interfaz para una experiencia de usuario mejorada.

## Características
- Página de login con campos para usuario y contraseña.
- Avatar personalizado en SVG.
- Diseño responsivo que se adapta a móviles y desktop.
- Estilos sobrios en azul petróleo con gradientes y efectos visuales.
- Interfaz de usuario intuitiva, moderna y centrada.

## Instalación
1. Clona este repositorio.
2. Abre el archivo `login.html` en tu navegador web.

## Tecnologías Utilizadas
- HTML
- CSS
- SVG
